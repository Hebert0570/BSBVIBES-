# BSbVibes

## 📌 Objetivo do Projeto

O projeto teve como objetivo desenvolver um protótipo funcional utilizando abordagens **Low Code / No Code / Vibecode**, explorando ferramentas capazes de acelerar o desenvolvimento de aplicações sem exigir programação tradicional em larga escala.  

A proposta escolhida pelo grupo foi a criação do **BSbVibes**, um portal de turismo inclusivo focado no mapeamento e detalhamento de acessibilidade dos principais monumentos da Esplanada em Brasília:

* **Museu Nacional**
* **Catedral Metropolitana**
* **Congresso Nacional**

A solução demonstra fluxos interativos de navegação, uma experiência de usuário (UI/UX) limpa e a superação de barreiras técnicas de plataformas gratuitas por meio de injeção de scripts customizados.
